# SwiftStick

A minimalist iOS gamepad controller for Mac.

## Permissions

- **Local Network**: Required to discover and connect to your Mac via MultipeerConnectivity.

## How to Build

1. Open `SwiftStick.xcodeproj` in Xcode.
2. Select the target device (iOS for Controller, Mac for Receiver).
3. Run the application.
